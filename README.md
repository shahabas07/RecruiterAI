# RecruiterAI

RecruiterAI is an advanced, AI-native Application Tracking System (ATS) and recruiting platform designed to streamline the hiring process. By leveraging artificial intelligence for sourcing, screening, and scheduling, RecruiterAI helps teams hire the top 1% of talent in minutes.

## Features

- **Automated Sourcing**: Our AI engine identifies the best candidates across multiple platforms.
- **Smart Screening**: Instantly parse and rank resumes based on job requirements, eliminating manual review fatigue.
- **Global Hiring Support**: Built-in tools for managing international teams and compliance.
- **Seamless Scheduling**: Automated interview coordination to reduce time-to-hire.
- **Data-Driven Insights**: Real-time analytics on your hiring pipeline and candidate quality.

## Tech Stack

- **Frontend core**: React 18, Vite
- **UI Components**: Radix UI primitives for accessible and robust interactive elements.
- **Styling**: Modern CSS with utility classes.
- **Icons**: Lucide React.
- **Animations**: CSS animations for smooth user interactions.

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/shahabas07/RecruiterAI.git
   ```
2. Navigate to the project directory:
   ```bash
   cd RecruiterAI
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development

To start the development server:

```bash
npm run dev
```

The application will be available at `http://localhost:3000`.

### Build

To create a production build:

```bash
npm run build
```
